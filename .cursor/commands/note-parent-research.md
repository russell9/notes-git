# Родительская заметка: исследование + BibTeX (только первый проход)

Выполни для **текущей открытой** или **указанной** заметки — только если она подходит под условия ниже.

## Кому применять (обязательная проверка перед любыми правками)

1. **Только родительские (корневые) заметки**  
   В YAML должно быть **`parents: []`** или поле **`parents`** отсутствует.  
   Если в `parents` есть **хотя бы один** путь к файлу — это **дочерняя** заметка → **ничего не меняй**, напиши пользователю: «Пропуск: дочерняя заметка».

2. **Только если пайплайн ещё не запускали**  
   Если во frontmatter уже есть **`hub_research_applied: true`** → **ничего не меняй**, напиши: «Уже обработано».

Если любая проверка не пройдена — **остановись** и не редактируй файл.

## Шаги (только после успешных проверок)

1. **`# @context`**  
   Кратко зафиксируй исходную постановку: **title** заметки + суть раздела «Контекст» / черновика «Основное», если они уже есть (это **собственный** контекст файла, не подтягивай другие заметки как родителя).

2. **`## @goal`**  
   Сформулируй цель исследования по **title** и содержанию этой же заметки.

3. **`# Основное`**  
   Добавь или дополни структурированным исследованием: обзор литературы, смежные области, перспективы; ссылайся на проверяемые источники (arXiv, proceedings, журналы).

4. **`# Ссылки`**  
   Добавь блок ` ```bibtex ` с записями; ключи в формате `фамилияГодкратко`.

5. **Общий bib**  
   Если в vault есть `notes/projects/projects.bib` (или аналогичный путь от корня открытого workspace), добавь туда **только новые** уникальные записи (без дубликатов ключей).

6. **Пометка о завершении**  
   В YAML frontmatter добавь или обнови:
   - `hub_research_applied: true`
   - `hub_research_applied_at: YYYY-MM-DD` (сегодняшняя дата)

## Ограничения

- Не удаляй без запроса уже написанный осмысленный текст; дополняй согласованные разделы.
- **Не** используй эту команду для ветвления «дочерних» заметок — для них отдельный сценарий (если появится).

После выполнения кратко перечисли изменённые файлы.
